# 数据库表设计说明

> 完整 DDL 见 **`backend/doc/schema.sql`**；增量迁移见 **`backend/doc/migration-20250317.sql`**。

## 1. 核心实体关系（简述）

```
sys_user ──┬── course (teacher_id)
           ├── course_student
           ├── question ── knowledge_point
           ├── paper ── paper_question ── question
           ├── exam ── exam_record ── exam_answer
           └── wrong_book
```

## 2. 主要表（按业务）

### 用户与权限

| 表 | 说明 |
|----|------|
| `sys_user` | 用户、角色（ADMIN/TEACHER/STUDENT）、密码 |
| `sys_menu` | 菜单、路径、`perms` 与接口权限对齐 |
| `sys_role_menu` | 角色与菜单多对多 |

### 课程与题库

| 表 | 说明 |
|----|------|
| `course` | 课程、授课教师 |
| `course_student` | 选课关系 |
| `knowledge_point` | 知识点树（含 `parent_id`） |
| `question` | 试题（题型、难度、选项 JSON、答案等） |

### 试卷与组卷

| 表 | 说明 |
|----|------|
| `paper` | 试卷（MANUAL/AUTO、`rules_json` 记录自动组卷规则） |
| `paper_question` | 试卷题目顺序与分值 |

### 考试与作答

| 表 | 说明 |
|----|------|
| `exam` | 考试场次、时间、及格线、成绩发布、乱序与切屏策略等 |
| `exam_record` | 学生答卷、乱序种子、切屏次数等 |
| `exam_answer` | 作答明细 |

### 其他

| 表 | 说明 |
|----|------|
| `file_resource` | 上传文件 |
| `wrong_book` | 错题本 |

## 3. 题型与难度

- **题型**：`question.type` 枚举见 `QuestionTypeEnum`（如 SINGLE、MULTIPLE、TRUE_FALSE、FILL、SHORT）。
- **难度**：`question.difficulty`：`1` 易、`2` 中、`3` 难（可空，组卷时按易档处理）。

## 4. 索引与约束

- 外键与唯一约束见 `schema.sql`；生产环境可评估是否需要按查询场景增加 `course_id`、`exam_id` 等组合索引。
