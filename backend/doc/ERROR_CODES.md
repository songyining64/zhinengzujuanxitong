# 错误码说明

业务异常通过 `BizException` 携带 `ErrorCode`，HTTP 状态与 `code` 字段对应关系如下：

| code | 含义 |
|------|------|
| 0 | 成功 |
| 40000 | 请求参数不合法 |
| 40001 | 参数校验失败 |
| 40002 | 分页参数越界 |
| 40100 | 未登录或令牌无效 |
| 40300 | 无权限 |
| 40400 | 资源不存在 |
| 40900 | 数据约束或状态冲突 |
| 40901 | 重复数据（唯一约束等） |
| 50000 | 服务内部错误 |

前端可根据 `code` 与 `msg` 做统一提示；`40901` 常见于重复提交或并发抢占唯一资源。
